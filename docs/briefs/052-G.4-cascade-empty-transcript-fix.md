# /tdd brief — cascade_empty_transcript_overfail_fix

> **Real-key-smoke bug-fix — THE ACTIVE cascade blocker (pulled forward from G.4-candidate).** With the model fix (051) live-confirmed, the user's cascade turn now fails on **`cascade.empty_transcript`**: a leading-silence/VAD-boundary **empty STT final fails the WHOLE turn**. The orchestrator (`CascadeStreamingOrchestrator.RunAsync`, ~L164-169) fails on the **FIRST** empty/whitespace final (`Error(empty_transcript) + Done(Failed) + yield break`) — it never skips a spurious empty final or waits for a real one. The inverse of the §28 join-finals lesson. **This IS a `/tdd` slice** (deterministic orchestrator logic). The backend impl diagnosed this while doing 048; the fix shape is its own.

## Feature
Make a cascade turn tolerate spurious empty/whitespace STT finals: **skip** empty finals (don't translate, don't fail), proceed on the first **non-empty** final (the normal path), and fail with `cascade.empty_transcript` **only if NO non-empty final ever arrives** (at stream end).

- **Architecture:** `ARCH-011` (cascade streaming — STT→translation handoff), `ARCH-018` (fail-closed only when genuinely empty).
- **Evidence (live turn, post-051):** source transcript EMPTY; `stt.final` fired (~1121 ms) but empty; UI "cascade request failed"; turn status `failed`. (A leading pause produced an empty final → the turn died.)

## Root cause (impl-diagnosed, confirm via the RED test)
`CascadeStreamingOrchestrator.RunAsync` `SttFinal` case (~L164-169): `if (string.IsNullOrWhiteSpace(final.Text)) { yield Error(empty_transcript); yield Done(Failed); yield break; }` — fails the whole turn on the **first** empty/whitespace final, never skipping it or awaiting a later non-empty one.

## Fix
- **Skip** an empty/whitespace `SttFinal` (do NOT translate it, do NOT fail the turn) — continue consuming the STT stream.
- Proceed normally on the first **non-empty** final (the existing translate→TTS path — unchanged).
- Fail with `cascade.empty_transcript` **only if the STT stream completes with NO non-empty final ever seen** (track a "saw a non-empty final" flag; fail-closed at stream end if false).
- Keep `IsNullOrWhiteSpace` as the empty test.
