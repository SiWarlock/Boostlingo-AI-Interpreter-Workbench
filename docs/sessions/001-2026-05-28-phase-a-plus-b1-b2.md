## Decisions made (with rationale)

- **Inline Options defaults = single source of truth** (appsettings carries NO provider defaults) — avoids the inline-vs-config divergence trap; env vars are the override path (ARCH-028).
- **Nulls written explicitly** (`"summary": null`, no `WhenWritingNull`) — verified against the ARCH-016 persisted example.
- **DateTimeOffset serializes ISO-8601 with explicit `+00:00`** (System.Text.Json default), not the `Z` shorthand — both valid; avoids a lossy UTC-forcing converter. (Orchestrator annotated ARCH-009/013/016.)
- **Round-trip equality asserted on JSON strings**, not record `==` (C# record equality is reference-based over `List`/`Dictionary` members).
- **Pricing `RealtimePricing` is an explicit class, not a dict** — the ARCH-014 `realtime` block has `estimatorNote` (a string) as a sibling of the model entries; a pure dict would throw. Translation/tts stay model-keyed dicts. (JSON kept verbatim.)
- **WAE scope = Api-only** (Tests relaxed) — test-SDK/xUnit analyzer warnings under WAE are pure friction. No `Directory.Build.props`. (Carry-forward closed.)
- **Swagger kept, Development-only** (registration + middleware both gated; prod-404 regression test).
- **Provider error `SafeMessage` = fixed generic per code, never `ex.Message`** — boundary safety (ARCH-018/019); the full sanitizer is B.8.
- **Fake error variants use a real ARCH-012 code** (`<stage>.upstream_unavailable`), not an invented `*.failed`.

## How to use what was built

- **Run the backend:** `cd server/AiInterpreter.Api && dotnet run` → `http://localhost:5179` (Swagger at `/swagger` in Development; health at `/api/health`). Keyless — providers aren't wired yet (fakes/real land in C.4).
- **Tests:** `dotnet test server/AiInterpreter.sln` (50 tests). `dotnet format --verify-no-changes` + `dotnet build` (WAE) for the quality gate.
- **Add a provider impl (C):** implement `ISttProvider`/`ITranslationProvider`/`ITtsProvider`; catch SDK/HTTP exceptions → `ProviderErrorMapper.Map(ex, provider, stage)`; stream events via `async IAsyncEnumerable` + `[EnumeratorCancellation]` (see the B.2 fakes for the pattern).
- **Resume at B.3** (latency model + `MetricsAggregator`): inject `IClock` for deterministic time; metrics consume the domain `LatencyEvent` (A.3).
