## Decisions made

- **B.3:** aggregator computes from **absolute `Timestamp` subtraction** (cross-clock safe), not per-event `relativeMs`; the **factory** clamps a single event's `relativeMs` â‰¥ 0 (monotonic vs same-clock origin) but the **aggregator does NOT clamp** cross-clock pairs (skew disclosed, not hidden). Under-specified ARCH-013 origins resolved by convention: cascade stage-duration metrics measure from the stage-start origin; realtime origins (`realtime_connect_ms` â†گ new `realtime.session.connecting`; `realtime_playback_ms` = playback âˆ’ first_audio_delta).
- **B.4:** nested per-segment loop (sequential per finalized STT segment) â€” NOT consume-all-STT-then-translate (which would buffer a multi-segment utterance = the full-utterance blocking ARCH-011 forbids). Try/catch only around `MoveNextAsync` (C# forbids yield-in-try); OCE filtered `when (!ct.IsCancellationRequested)` to split caller-cancel from stage-timeout; STT stage uses a **per-event idle timeout** (armed per `MoveNextAsync`, disarmed during downstream â€” `CancelAfter` can't un-cancel a fired CTS). Timeouts as `TimeSpan?` (defaults 30/20/30s). Target partials carry cumulative accumulated text.
- **B.5:** cascade turn â†’ one composite `CostEstimate` (`Provider="cascade"`, `Model`=translation model = the comparison axis, `PricingBasis="composite"`); `decimal`, no rounding; a `0.0` configured rate estimates to 0.0 (not "unavailable" â€” only genuinely-absent config degrades); realtime audio-secâ†’token factor is an explicit estimate (`RealtimeTokensPerAudioSecond = 50`, build-confirm).
- **B.6:** punctuation **removed** (replaced with "", `\p{P}` only) not spaced; accents **preserved**; S/I/D tie-break precedence **match > substitution > deletion > insertion**; empty reference is a **precondition** (`ArgumentException`) not a degradable state; WER unbounded (>1.0 valid, never clamp); phrase store is a self-loading instance facade over an internal `Result`-returning load.
